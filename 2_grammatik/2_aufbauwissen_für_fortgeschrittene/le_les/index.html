Was sind indirekte Objektpronomen?

Indirekte Objektpronomen im Spanischen, wie le und les, verwendet man, um zu sagen, wem etwas passiert oder für wen eine Handlung bestimmt ist. Sie stehen anstelle von Personen oder Dingen, um das Sprechen einfacher und schneller zu machen.

Die indirekten Objektpronomen:

Hier sind die häufigsten indirekten Objektpronomen im Spanischen:

Person Pronomen Beispiel Deutsche Übersetzung
yo me Juan me dio un regalo. Juan gab mir ein Geschenk.
tú te Te envié una carta. Ich schickte dir einen Brief.
él/ella/usted le Le conté una historia. Ich erzählte ihm/ihr eine Geschichte.
nosotros nos Nos dieron un regalo. Sie gaben uns ein Geschenk.
vosotros os Os enviaron una invitación. Sie schickten euch eine Einladung.
ellos/ellas/ustedes les Les dije la verdad. Ich sagte ihnen die Wahrheit.

Beispiele für die Verwendung:

Ich gab ihm das Buch.
Auf Spanisch: Le di el libro.

Sie schickten uns eine Postkarte.
Auf Spanisch: Nos enviaron una postal.

Er erzählte ihnen eine Geschichte.
Auf Spanisch: Él les contó una historia.
